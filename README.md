# ShroomIT
## Overview
ShroomIT is a simple automated mushroom growing project

## Components
* Teensy 4.0
* DHT11 sensor 
* Breadbord, resistors, jumper wires
* LCD display
* 4-channel Relay
* LEDs
* Ultrasonic atomiser
* Fan (optional: potentiometer)
